## Instalace

1. **Venv:**

```bash
python -3 -m venv venv
source ./venv/Scripts/activate
```

2. **Instalace Knihoven:**
```bash
pip install -r requirements.txt
```

3. **Složka Projektu:**
```bash
cd gaflix
```

4. **Spuštění:**
```bash
python manage.py runserver
```
